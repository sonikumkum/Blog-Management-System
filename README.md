# Blog-Management-System
A simple Blog Management System built using MERN stack with CRUD functionality to create, edit, delete, and view blog posts.

It allows users to create, read, update, and delete (CRUD) blog posts.

## Features
- Add new blog posts
- Edit existing posts
- Delete posts
- View all blogs

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
